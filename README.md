# Awesome Torah Tech

Collection of technological projects for religious Jewish use. This includes Hebrew calendar utilities, tools for studying Jewish texts, AI-powered Torah tools, and other technology intended to make life easier for religiously observant Jews.

## Disambiguation

This list focuses on projects specifically intended for religiously observant Jews, rather than general Israel/Israeli tech projects. Projects are included without regard to the specific type of Judaism practiced by, or envisioned by, the project maintainers.
